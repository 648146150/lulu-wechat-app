# 工具箱 · 微信小程序

一个集成了记账、资产、步数、经期记录的个人生活管理工具箱。

---

## 功能模块

| 模块 | 说明 |
|------|------|
| 📊 记账本 | 收支记录、按月统计、分类排行、长按删除 |
| 💰 资产记录 | 多账户汇总、外汇币种选择、实时汇率换算为人民币净资产 |
| 🚶 步数记录 | 微信运动解密、7 天柱状图、目标进度 |
| 📅 经期记录 | 日历标记、周期预测、距离下次倒计时 |
| ⚙️ 卡片管理 | 仪表盘卡片槽位增删替换，最多 6 槽位 |

---

## 技术栈

- 微信小程序原生框架
- 高德天气 API（自动 GPS 定位，失败回退北京）
- 免费汇率 API（open.er-api.com，支持 11 种币种）
- 微信云开发（步数解密云函数）
- 本地存储（wx.Storage，无后端依赖）

---

## 快速开始

1. 微信开发者工具导入项目
2. 注册 [高德开放平台](https://lbs.amap.com) 获取 Web 服务 Key，填入 `utils/config.js`
3. 开通微信云开发，获取环境 ID，填入 `app.js`
4. 部署云函数 `cloudfunctions/getWeRunData`
5. 编译运行

## 项目结构

```
├── app.js / app.json / app.wxss  # 全局入口与样式
├── pages/
│   ├── dashboard/                # 仪表盘（时间+天气+工具卡片网格）
│   ├── bookkeeping/              # 记账本（index/add/history）
│   ├── assets/                   # 资产记录（账户列表+外汇汇率）
│   ├── steps/                    # 步数记录（云函数解密）
│   ├── period/                   # 经期记录（日历+周期预测）
│   └── settings/                 # 卡片槽位管理
├── utils/
│   ├── storage.js                # 账单数据层
│   ├── weather.js                # 天气（高德 API）
│   ├── exchange.js               # 汇率（实时+缓存）
│   ├── assets-storage.js         # 资产数据层（外汇汇算）
│   └── period-storage.js         # 经期数据层
├── cloudfunctions/getWeRunData/  # 微信运动解密云函数
└── preview/index.html            # UI 设计预览
```
