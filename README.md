# 记账本 · 微信小程序

一个极简的个人记账小程序，支持收入/支出记录、按月统计、分类排行、自动定位天气。

## 📊 数据设计

### 单条账单记录
```js
{
  id: "bill_xxx_xxx",           // 唯一标识
  type: "expense" | "income",   // 支出/收入
  amount: 50.00,                // 金额
  category: "餐饮",             // 分类名称
  note: "午饭",                 // 备注（选填）
  date: "2026-06-19",          // 日期
  createTime: "2026-06-19T..."  // 创建时间戳
}
```

### 内置分类

| 类型 | 分类 |
|------|------|
| 支出 | 🍽️餐饮 🚗交通 🛍️购物 🎮娱乐 🏠居家 💊医疗 📚教育 💸其他 |
| 收入 | 💰工资 🎁奖金 💼兼职 📈理财 ↩️退款 📥其他 |

## 🎨 设计要点

- **配色**：收入绿色（#4CAF50），支出红色（#F44336），天气蓝色渐变
- **导航**：自定义底部三栏（🏠首页 · ➕记账 · 📋账单）
- **天气**：自动 GPS 定位，失败回退北京；高德 API 免费额度
- **存储**：`wx.setStorageSync` 本地存储，上限 10MB，无后端依赖
- **月份切换**：翻看历史月份，不翻未来；首页 + 账单页双重筛选

## 🚀 快速开始

1. 微信开发者工具导入项目目录
2. 注册 [高德开放平台](https://lbs.amap.com) 获取 Key
3. 复制 `utils/config.example.js` → `utils/config.js`，填入 Key
4. 编译运行
